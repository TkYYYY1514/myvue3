# Vue 3 项目技术栈配置

## 核心框架
- **前端框架**: Vue 3.5.31 + Vite 8.0.3
- **语言**: JavaScript (暂不使用 TypeScript)
- **UI组件**: Element Plus 2.13.6 (全局引入)
- **图标库**: Element Plus Icons (全局引入)
- **状态管理**: Pinia 3.0.4
- **路由**: Vue Router 5.0.4
- **网络请求**: Axios1.14. 0
- **Mock数据**: Mock.js 1.1.0
- **开发工具**: vite-plugin-vue-devtools 8.1.1
- **Node版本**: ^20.19.0 || >=22.12.0

## 技术亮点
1. **组合式API (Composition API)**: 使用Vue 3的组合式API编写组件
2. **状态管理**: 使用Pinia进行状态管理，支持setup语法糖
3. **组件通信**: 使用provide/inject实现跨组件通信
4. **路由管理**: 使用Vue Router进行路由管理，支持路由元信息和懒加载
5. **UI框架**: 使用Element Plus提供丰富的UI组件
6. **图标系统**: 全局注册Element Plus Icons，方便使用
7. **主题切换**: 支持亮色/暗色主题切换
8. **响应式布局**: 使用Element Plus的布局系统实现响应式设计
9. **路由动画**: 使用Vue的Transition组件实现路由切换动画
